# Лабораторна робота №8 — Файлове введення-виведення, серіалізація та винятки в Java

## Опис

У лабораторній роботі реалізовано:
- принципи об’єктно-орієнтованого програмування;
- наслідування та поліморфізм;
- використання інтерфейсів;
- файлове введення-виведення;
- серіалізацію об’єктів;
- обробку винятків;
- модульне тестування за допомогою TestNG.

Проєкт реалізує систему бронювання номерів готелю.

---

## Реалізовані класи

### Основні класи
- `Room` — абстрактний базовий клас номера;
- `StandardRoom`
- `FamilyRoom`
- `LuxuryRoom`
- `Guest`
- `Booking`
- `Hotel`

### Інтерфейси
- `Reservable`
- `Bookable`
- `Cleanable`

### Робота з файлами
- `HotelFileManager`

---

## Що було додано у лабораторній роботі №8

У порівнянні з лабораторною роботою №7 було додано:

### Серіалізацію
Класи були оновлені для підтримки серіалізації:
- `Room`
- `Hotel`

з використанням:

```java
implements Serializable
```

---

### Роботу з файлами
Створено клас:

```java
HotelFileManager
```

з методами:
- `saveHotel()`
- `loadHotel()`

Для цього використано:
- `ObjectOutputStream`
- `ObjectInputStream`

---

### Обробку винятків
Реалізовано перевірки для:
- некоректної ціни номера;
- некоректної кількості діб бронювання;
- некоректної місткості номера;
- порожніх ідентифікаторів та імен.

---

### Тестування
Додано тести для:
- серіалізації;
- обробки винятків;
- наслідування;
- поліморфізму;
- інтерфейсів.
