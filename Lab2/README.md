# Lab 2 — Вимоги та зв’язки між класами

## Опис

У даній роботі реалізовано систему бронювання номерів з урахуванням зв’язків між класами.

## Реалізовані зв’язки

- Hotel → Room (агрегація)
- Booking → Guest (асоціація)
- Booking → Room (композиція)

## Функціонал

- додавання номерів до готелю
- пошук вільного номера
- створення бронювання
- скасування бронювання
