Laboratory Work No.1

Object-Oriented Programming. Java
Variant 3: Hotel Room Booking System

Опис проєкту

У даній роботі реалізовано спрощену об’єктно-орієнтовану модель системи бронювання номерів у готелі.

Система дозволяє:

зберігати інформацію про номери
працювати з гостями
створювати та скасовувати бронювання
обчислювати вартість проживання
Предметна область

Система складається з трьох основних класів:

Клас Room

Описує номер у готелі.

Поля:

roomNumber — унікальний номер
roomType — тип номера
pricePerNight — ціна за ніч
available — доступність

Методи:

book() — робить номер зайнятим
release() — звільняє номер
Клас Guest

Описує гостя.

Поля:

guestId — унікальний ідентифікатор
fullName — ім’я
phone — номер телефону (необов’язковий)

Методи:

hasPhone() — перевірка наявності телефону
Клас Booking

Описує бронювання.

Поля:

bookingId — унікальний ідентифікатор
guest — гість
room — номер
nights — кількість ночей
active — стан бронювання

Методи:

totalCost() — обчислення вартості
cancel() — скасування бронювання
Основні можливості
використано інкапсуляцію (private поля)
реалізовано перевантаження конструкторів
перевизначено методи:
toString()
equals()
hashCode()
реалізована логіка предметної області:
автоматичне бронювання номера
скасування бронювання
розрахунок вартості

Висновок

У даній лабораторній роботі було розроблено об’єктно-орієнтовану модель системи бронювання номерів у готелі.

Було застосовано основні принципи ООП:

інкапсуляція
перевантаження конструкторів
перевизначення методів

Розроблену систему можна розширити в майбутньому, додавши нові можливості, такі як управління готелем, оплата та робота з датами.
