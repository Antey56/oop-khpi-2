# Звіт до лабораторної роботи №13

## Тема

Обробка тексту і регулярні вирази в Java.

## Предметна область

За основу взято систему бронювання готельних номерів із попередніх лабораторних робіт. У цій роботі вхідними даними є текстові підтвердження бронювань. Кожен рядок містить дату створення запису, код бронювання, клієнта, телефон, номер кімнати, тип кімнати, ціну, дати заїзду/виїзду та статус.

## Вхідний файл

Файл `src/main/resources/booking-confirmations.txt` містить коректні й некоректні записи. Програма читає його через `Files.readAllLines()`.

## Текстова обробка

У класі `BookingTextProcessor` реалізовано повний ланцюг обробки:

1. очищення рядка через `trim()`;
2. нормалізація пробілів через `replaceAll("\s+", " ")`;
3. перевірка формату через `Pattern`;
4. вилучення полів через `Matcher.group()`;
5. перетворення рядкових значень у `LocalDate`, `double` та об'єкти предметної області;
6. формування звіту через `StringBuilder`.

## Регулярний вираз

Шаблон у `BookingTextProcessor` перевіряє формат коду `BOOK-R-001`, ідентифікатора клієнта `C-001`, телефону, номера кімнати, типу кімнати, ціни, дат і статусу. Групи захоплення використовуються для створення об'єктів `Reservation`, `Customer` і `Room`.

## StringBuilder

Метод `buildReport()` формує багаторядковий звіт: кількість рядків, кількість валідних записів, кількість помилкових рядків, активний дохід і перелік розібраних бронювань.

## Тестування

У `BookingTextProcessorTest` перевірено:

- нормалізацію пробілів;
- валідацію коду бронювання;
- розбір коректного рядка;
- відхилення некоректного рядка;
- обробку скасованого бронювання;
- формування текстового звіту.

## Висновок

У роботі було реалізовано прикладну текстову обробку в Java. Було показано, як сирий текст із файлу перетворюється на структуровані об'єкти предметної області, як регулярні вирази допомагають перевіряти формат і вилучати поля, а `StringBuilder` використовується для побудови підсумкового звіту.
